# Changesets

This folder is managed by [changesets](https://github.com/changesets/changesets).

- Run `pnpm changeset` to record an intent-to-release for any changed `@platform/*` package.
- Run `pnpm version-packages` to apply pending changesets (bump versions + write changelogs).
- Apps (e.g. `storefront`) are private and ignored — only shared packages are versioned.

See the [common questions](https://github.com/changesets/changesets/blob/main/docs/common-questions.md).
